# c18-backend

C18 Backend Repositorio

### Instrucciones

Una ves tenga docker instalado correr el siguiente comando:
`docker-compose up -d`
En caso de hacer cambios al proyecto correr el siguiente comando:
`docker-compose up --build -d`
